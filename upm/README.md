# MSP Unity SDK

Unity Package Manager plugin (`ai.themsp.unity`) for MSP ads.

Monorepo layout: see [../README.md](../README.md).

## Scope (MVP)

- Interstitial only
- Keep MSP public API semantics unchanged:
  - `Initialize`
  - `LoadAd`
  - `GetAd`
  - `Show`
